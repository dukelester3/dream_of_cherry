個人簡介

編輯首頁「關於悠悠」區塊的照片與文字。儲存後點左側「返回網站（預覽）」即時查看；正式發布請至「匯出資料」生成 data.js 並推送到 GitHub。

設定

用於後台上傳照片。免費註冊 imgbb.com 取得 Key，取得後貼上並儲存。
若未設定，上傳時可選「Base64 嵌入」（無需 API，但 data.js 會變大），或直接貼上圖片網址。

用於日記／客評編輯時一鍵翻譯。專案 gen-lang-client-0879457781。免費額度：約 250 次/日。

用於後台「發布到 GitHub」一鍵更新 data.js。至 GitHub Settings → Tokens 建立，勾選 repo 權限。

浮水印工具

上傳圖片後自動加上浮水印，可下載使用。與女孩/客評/日記上傳時使用的浮水印相同（不透明、右下角略偏左、較小）。

📷 點擊或拖曳圖片到此 支援 JPG、PNG 等

點擊圖片可放大

匯出資料

將後台修改的內容匯出為 data.js 檔案。要讓無痕/其他裝置看到更新，必須點「發布到 GitHub」（需先在設定填寫 Token）。

多人協作：發布時會自動合併 GitHub 上的最新資料，不會覆蓋他人新增的內容。建議發布前先點「從 GitHub 載入最新」取得遠端資料。
